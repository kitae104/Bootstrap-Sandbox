Bootstrap 5 간단한 스타터

이것은 Bootstrap 5 프로젝트를 만들고 사용자 정의하기 위한 보일러플레이트입니다.

"node_modules/bootstrap/dist/scss/_variables.scss"를 확인하여 "scss/bootstrap.scss" 파일 에 덮어쓸 수 있는 변수를 확인하세요 . 사용자 지정 스타일의 경우 "scss/styles.scss" 파일에 추가하세요. CSS 파일을 직접 편집하지 마세요.

다음 명령을 사용하여 Sass 파일을 빌드하거나 감시하세요.: 

npm run sass:build
npm run sass:watch
